Lyrics API - Genius

Introduction

The Lyrics API allows you to fetch song lyrics easily from Genius. You can use it to integrate lyrics into your applications.

Getting Started

Welcome to the Lyrics API! Our service offers a comprehensive collection of song lyrics from Genius. Whether you're building a music app, website, or just curious about song lyrics, our API has you covered.

The best part? Our Lyrics API is completely free to use! However, for accessing lyrics from Genius, you'll need to sign up for a Genius API key from the Genius Website.

Get started with our API now and explore our documentation to discover the available endpoints and start integrating lyrics into your projects today.

Supported Sources

The API fetches lyrics from Genius:

  • Genius: Good for lyrics and annotations from various music sources.

Endpoints:

Genius

Search Lyrics

Endpoint: /genius/lyrics

Method: GET

Parameters:

  • title (string): The search query (song title).
  • api_key (string): Genius API Key.

Example Request:

GET /genius/lyrics?title={song_title}&api_key={genius_key}
