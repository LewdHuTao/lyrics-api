Lyrics API - Musixmatch

Introduction

The Lyrics API allows you to fetch song lyrics easily from Musixmatch. You can use it to integrate lyrics into your applications.

Getting Started

Welcome to the Lyrics API! Our service offers a comprehensive collection of song lyrics from Musixmatch. Whether you're building a music app, website, or just curious about song lyrics, our API has you covered.

The best part? Our Lyrics API is completely free to use! No API key is required for this endpoint.

Get started with our API now and explore our documentation to discover the available endpoints and start integrating lyrics into your projects today.

Supported Sources

The API fetches lyrics from Musixmatch:

  • Musixmatch: Extensive lyrics library and sync capabilities.

Endpoints:

Musixmatch

Search Lyrics

Endpoint: /musixmatch/lyrics

Method: GET

Parameters:

  • title (string): The search query (song title).

Example Request:

GET /musixmatch/lyrics?title={song_title}

Example Response:

{
    "artist_name": "The Weeknd",
    "track_name": "After Hours",
    "track_id": 194169155,
    "search_engine": "Musixmatch",
    "artwork_url": "https://s.mxmcdn.net/images-storage/albums2/7/2/8/7/3/3/48337827_350_350.jpg",
    "lyrics": "Thought I almost died in my dream again (baby, almost died) ..."
}

For More Accurate Lyrics Results

Endpoint: /musixmatch/lyrics-search

Method: GET

Parameters:

  • title (string): The search query (song title).
  • artist (string): The artist's name for the song.

Example Request:

GET /musixmatch/lyrics-search?title={song_title}&artist={artist_name}

Example Response:

{
    "artist_name": "Rihanna",
    "track_name": "Diamonds",
    "track_id": 84458341,
    "search_engine": "Musixmatch",
    "artwork_url": "https://s.mxmcdn.net/images-storage/albums2/1/0/4/7/5/4/40457401_350_350.jpg",
    "lyrics": "Shine bright like a diamond ..."
}

Response Codes

  • 200: Success
  • 400: Bad Request
  • 401: Unauthorized
  • 404: Not Found
  • 500: Internal Server Error
