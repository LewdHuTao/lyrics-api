Lyrics API - YouTube Music

Introduction

The Lyrics API allows you to fetch song lyrics easily from YouTube Music. You can use it to integrate lyrics into your applications.

Getting Started

Welcome to the Lyrics API! Our service offers a comprehensive collection of song lyrics from YouTube Music. Whether you're building a music app, website, or just curious about song lyrics, our API has you covered.

The best part? Our Lyrics API is completely free to use! No API key is required for this endpoint.

Get started with our API now and explore our documentation to discover the available endpoints and start integrating lyrics into your projects today.

Supported Sources

The API fetches lyrics from YouTube Music:

  • YouTube Music: Great for lyrics from YouTube videos.

Endpoints:

YouTube Music

Search Lyrics

Endpoint: /youtube/lyrics

Method: GET

Parameters:

  • title (string): The search query (song title).

Example Request:

GET /youtube/lyrics?title={song_title}

Example Response:

{
    "artist_name": "Rick Astley",
    "track_name": "Never Gonna Give You Up",
    "search_engine": "YouTube",
    "artwork_url": "https://lh3.googleusercontent.com/eC9DfRcYSk4FE-fvDCJSu_4xsKdVMKxwmFTYFZwP8OqB7R4TKxAjKoR-Kp1lXeRi2WddPFYulSte4eW-=w120-h120-l90-rj",
    "lyrics": "Never gonna give you up, Never gonna let you down ..."
}

Response Codes

  • 200: Success
  • 400: Bad Request
  • 401: Unauthorized
  • 404: Not Found
  • 500: Internal Server Error
